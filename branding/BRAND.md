# C50 · Crunch AppSec — Brand Guide

- **Accent:** Code Crunch blue `#2463EB` (with deep `#1d4ed8`).
- **Voice:** rigorous, ethical, and defense-first. Always show the flaw, then the exploit *in the lab*, then the source-level fix — every attack exists to teach a detection and a defense. Authorization, scope, and legality are stated, not assumed. Nothing is magic; a vulnerability is a decision you can trace and reverse.
- **Icon motif:** a shield over a bracket `{ }` — code guarded at the source.
- **One-liner:** "Ship secure software — model threats, find flaws, and fix them at the source."
